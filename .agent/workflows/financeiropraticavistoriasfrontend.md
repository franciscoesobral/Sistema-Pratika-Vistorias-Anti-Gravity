description: Criar Tela/Componente (Frontend)
---

Atue como Especialista em Next.js e Tailwind.
Quero criar uma nova página no Dashboard.
1. Use o padrão "App Router" (`app/page.tsx`).
2. Use componentes do Shadcn/UI se disponível, ou Tailwind puro.
3. O design deve ser limpo, profissional ("Enterprise Clean").
4. Se precisar buscar dados, crie uma função `fetch` tipada que chama nosso Backend na porta 3333.
5. Inclua tratamento de estado de "Loading" e "Error".
